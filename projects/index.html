A word finder game. Each round, the player is given a set amount of time to find as many words from the jumble of letters at the top as they can. Designed for desktop as well as mobile devices.

The real goal of this game is to be second language vocab practice. The found words can be selected to see their translation into another language.

  • Ruby on Rails
    • The barest backend, just to serve the single page app, and the wordlist files
  • Javascript:
    • jQuery
    • Asynchronous JS (AJAX)
    • Backbone
  • CSS/SCSS
    • Custom Styling
    • Skeleton
  • Wordreference API for translations
