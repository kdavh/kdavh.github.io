G Sales

I love garage sales. I know there are many others like me, so to assist our collective search for sales, I made a service that displays garage sales by location on a map, with optional filters for days and times, and favorited / unfavorited status.

The sale information is taken from craigslist, but the could be grown by allowing users to create their own sales.

  • Ruby on Rails
    • Devise authentication with custom form views
    • Model associations
    • Geocoder gem for integrating Google Maps API with ActiveRecord
    • Nokogiri gem for web parsing; extensive regular expressions
  • Javascript:
    • jQuery
    • Asynchronous JS (AJAX)
    • Backbone; Backbone relational
  • CSS/SCSS
    • Bootstrap

L Words

A word finder game. Each round, the player is given a set amount of time to find as many words from the jumble of letters at the top as they can. Designed for desktop as well as mobile devices.

The real goal of this game is to be second language vocab practice. The found words can be selected to see their translation into another language.

  • Ruby on Rails
    • The barest backend, just to serve the single page app, and the wordlist files
  • Javascript:
    • jQuery
    • Asynchronous JS (AJAX)
    • Backbone
  • CSS/SCSS
    • Custom Styling
    • Skeleton
  • Wordreference API for translations
